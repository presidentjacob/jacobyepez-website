Projects

Overload Digest - Python News Scraper & GUI Application

A Python-based GUI application that aggregates news from multiple sources and presents it in a continuously scrolling newsfeed. Built using Tkinter, Requests, BeautifulSoup, and Selenium, the app automatically scrapes political and tech articles from sources like CNN, Fox News, NPR, TechCrunch, and BBC.

Key Features:

  • Dynamically scrapes articles from multiple sources, handling both static and dynamically loaded content.
  • Threaded background scraping ensures the GUI remains responsive while updating content in real-time.
  • Auto-scrolling newsfeed simulates a modern news app experience.
  • Respects robots.txt and uses user-agent headers for ethical scraping.
  • Modular design allows easy addition of new news sources.

ROM Launcher - Tauri2.0, TypeScript, and Python Application

A multi-language application (Python, TypeScript, Rust, HTML) made with a peer that uses Tauri2.0 to create an application that can automatically scan for ROMs in a users files and run them using the appropriate emulator from one instance.

Key Features:

  • User memory, remembers username and games already scanned.
  • Uses a Twitch API to collect information on a game such as box art.
  • Links games to their respective emulator to launch a game.
